1.はじめに
　この度は、World Timepiece 4.0をダウンロードしていただきありがとうございます。
　これはアルファ版であり、完成版ではないのでご了承ください。
　また、アルファ版につき、動作を保証することは出来ません。

2.使用方法
　実行ファイルは同梱していません。
　World_Timepiece_4.0_Source.hspをコンパイルしてお使いください。
　
　コンパイルには「HSP スクリプトエディタ」が必要です。
　World_Timepiece_4.0_Source.hspをHSPスクリプトエディタで開いてF5キーを押すか、
　メニューバーの「HSP(P)」→コンパイル(C)をクリックすると実行できます。

3.操作方法
　今のところ東京の現地時刻のみ表示できます。
　
　都市名が書かれている緑色のバーの横に、「＜」と書かれたボタンがあります。
　そのボタンを押すと、都市選択メニューが開かれます（今のところ未実装ですが）。
　
　Walls Luxor以降のWallsやDarwal OSで利用される際は、
　WallsまたはDarwal OSのエクスプローラーから
　　「World_Timepiece_4.0.ax」
　を選択して起動してください。
